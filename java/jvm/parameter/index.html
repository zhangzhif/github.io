java常用启动参数

2020-08-01

运行时数据区

参数 说明 示例
-Xss 虚拟机栈大小 ‐Xss512k
-Xms 堆的初始大小 -Xms1024m
-Xmx 堆可到达的最大值 -Xms20484m
-Xmn 堆中新生代的大小 -‐Xmn1024M
-XX:MaxMetaspaceSize 元空间最大值, 默认是-1,即不限制或者,只受限于本地内存大小 ‐XX:MaxMetaspaceSize=256M
-XX:MetaspaceSize 元空间触发Fullgc的初始阈值,单位为字节, 默认21M ‐XX:MetaspaceSize=256M
